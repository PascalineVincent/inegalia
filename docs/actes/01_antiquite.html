Antiquité · –350 av. J.-C. → 500 ap. J.-C.

Entrée du journal · Athènes, –350 av. J.-C.

Je suis arrivé dans une ville de 300 000 âmes, dont 100 000 esclaves. Personne ne les compte dans les assemblées. Le philosophe que j’ai rencontré m’explique que c’est dans l’ordre des choses — certains naissent pour commander, d’autres pour obéir. Il appelle ça la nature. Moi, j’appelle ça un choix.

Comment l’inégalité est perçue

L’inégalité antique est ontologique : elle précède l’économie. Elle n’est pas le résultat d’un système — elle est la condition du système. Aristote distingue l’oikonomia (gestion raisonnable du foyer) de la chrématistique (accumulation sans limite), qu’il condamne comme contre-nature. Paradoxe fondateur : la même philosophie justifie l’esclavage et critique l’avidité.

Comment elle est justifiée

Penseur Justification Angle mort
Aristote Hiérarchies naturelles L’esclave comme “instrument animé”
Platon Mythe des métaux Égalité réservée aux gardiens
Épicure Sobriété volontaire Ne s’adresse qu’aux hommes libres
Xénophon Gestion efficace Les femmes comme ressource

Code 
df <- tibble::tribble(
  ~societe,          ~gini, ~annee,
  "Athènes classique",  0.42, -400,
  "Rome républicaine",  0.43, -100,
  "Rome impériale",     0.49,  100,
  "Égypte romaine",     0.45,  200,
  "Byzance",            0.41,  600
) |> mutate(societe = forcats::fct_reorder(societe, annee))

p <- ggplot(df, aes(x = societe, y = gini,
                    text = paste0(societe, "<br>Gini : ", gini,
                                  "<br>ca. ", ifelse(annee<0, paste0(abs(annee)," av."), annee)))) +
  geom_col(fill = "#8B1A1A", alpha = .75, width = .6) +
  geom_hline(yintercept = .40, linetype = "dashed", color = "#B8860B", linewidth = .5) +
  annotate("text", x = .7, y = .41, label = "seuil d'extraction (Milanovic)",
           hjust = 0, size = 3, color = "#B8860B") +
  scale_y_continuous(limits = c(0, .65)) +
  labs(x = NULL, y = "Gini estimé",
       caption = "Source : Milanovic, Lindert & Williamson (2011)") +
  theme_minimal(base_size = 11, base_family = "serif") +
  theme(panel.grid.minor = element_blank(),
        panel.grid.major.x = element_blank())

plotly::ggplotly(p, tooltip = "text")

Les voix poétiques

Certains disent qu’une armée de cavalerie,
d’autres, une armée de fantassins,
est la plus belle chose sur cette terre noire.
Mais moi, je dis que c’est ce qu’on aime.

Sappho, Fragment 16, ca. 600 av. J.-C. — trad. Anne Carson

Pourquoi Sappho ici ? Son fragment pose une théorie de la valeur radicalement subjective, contre la logique de la force et de l’accumulation. Ce que vaut quelque chose, c’est ce qu’on désire — pas ce qu’on possède.

Fil rouge — Les inégalités

Comment l’inégalité est mesurée à cette époque : elle ne l’est pas. Il n’existe ni statistique, ni enquête, ni indicateur. L’inégalité est visible (corps des esclaves, palais des maîtres) mais non quantifiée. Le Gini que nous calculons rétrospectivement est une reconstruction — nos catégories projetées sur leur monde.

Fil femmes — Antiquité

Sappho est la première grande voix poétique féminine connue à théoriser le désir contre la logique économique de la force. Jane Marcet (1769–1858), première vulgarisatrice de l’économie politique, sera son héritière 2 400 ans plus tard — même invisibilité dans les manuels canoniques.
