Atelier 1 — Revenus vs patrimoine : le grand écart

Voir le code 
df <- tibble::tribble(
  ~pays,          ~gini_rev, ~gini_pat, ~region,
  "Danemark",     0.29, 0.64, "Europe Nord",
  "Suède",        0.27, 0.62, "Europe Nord",
  "Allemagne",    0.32, 0.67, "Europe",
  "France",       0.30, 0.65, "Europe",
  "Italie",       0.34, 0.61, "Europe Sud",
  "États-Unis",   0.39, 0.87, "Amériques",
  "Brésil",       0.53, 0.77, "Amériques",
  "Chine",        0.38, 0.71, "Asie",
  "Inde",         0.35, 0.75, "Asie",
  "Afrique du S.", 0.63, 0.82, "Afrique"
)

p1 <- ggplot2::ggplot(df, ggplot2::aes(gini_rev, gini_pat, color=region,
  text=paste0(pays,"<br>Revenus: ",gini_rev,"  Patrimoine: ",gini_pat))) +
  ggplot2::geom_abline(slope=1, intercept=0, linetype="dashed", color="#E2E8F0") +
  ggplot2::geom_point(size=4, alpha=.85) +
  ggplot2::geom_text(ggplot2::aes(label=pays), vjust=-1, size=2.8, family="sans") +
  ggplot2::scale_color_brewer(palette="Set2") +
  ggplot2::labs(x="Gini des revenus", y="Gini du patrimoine", color=NULL,
    caption="Sources : OCDE, WID.world · estimations 2022-2023") +
  ggplot2::theme_minimal(base_size=11) +
  ggplot2::theme(legend.position="bottom", panel.grid.minor=ggplot2::element_blank())

plotly::ggplotly(p1, tooltip="text")
Questions de l'atelier 1
Tous les points sont au-dessus de la diagonale. Qu'est-ce que cela signifie ? Pourquoi le patrimoine est-il toujours plus inégalement distribué que le revenu ? Les États-Unis sont-ils un cas particulier par rapport aux autres pays riches ?

Atelier 2 — La courbe de Kuznets tient-elle ?

Kuznets (1955) prédisait une courbe en cloche : les inégalités augmentent pendant l’industrialisation, puis diminuent spontanément avec le développement.

Voir le code 
df_k <- tibble::tibble(
  pib = c(500,1000,2000,4000,8000,15000,25000,40000),
  gini = c(.32,.40,.48,.52,.49,.43,.35,.31),
  label = c("Subsistance","Démarrage","Industrialisation","Pic",
             "Croissance","Tertiaire","Développé","Avancé")
)
p2 <- ggplot2::ggplot(df_k, ggplot2::aes(pib, gini,
    text=paste0(label,"<br>PIB/hab: $",pib,"<br>Gini: ",gini))) +
  ggplot2::geom_line(color="#2563EB", linewidth=1.2) +
  ggplot2::geom_point(color="#2563EB", size=3) +
  ggplot2::scale_x_log10(labels=scales::dollar_format()) +
  ggplot2::annotate("point", x=39000, y=.47, color="#DC2626", size=5, shape=17) +
  ggplot2::annotate("text", x=39000, y=.50, label="USA 2023", size=3, color="#DC2626") +
  ggplot2::annotate("point", x=13000, y=.38, color="#059669", size=5, shape=17) +
  ggplot2::annotate("text", x=13000, y=.41, label="Chine 2023", size=3, color="#059669") +
  ggplot2::labs(x="PIB/hab (log, $)", y="Gini estimé",
    caption="Schéma Kuznets + contre-exemples contemporains") +
  ggplot2::theme_minimal(base_size=11) +
  ggplot2::theme(panel.grid.minor=ggplot2::element_blank())

plotly::ggplotly(p2, tooltip="text")
Questions de l'atelier 2
Les États-Unis et la Chine (triangles) s'écartent de la courbe de Kuznets. Comment interpréter ces contre-exemples ? La baisse des inégalités peut-elle être automatique — ou faut-il des choix politiques ? Que disent les Trente Glorieuses à ce sujet ?

Atelier 3 — Comprendre r > g

Piketty montre que quand le rendement du capital r dépasse le taux de croissance g, les patrimoines s’accumulent plus vite que les revenus du travail. Voici les données historiques.

Voir le code 
df_rg <- tibble::tribble(
  ~periode,            ~r,   ~g,
  "Moyen Âge",         4.5,  0.2,
  "18e siècle",        5.0,  0.5,
  "19e siècle",        5.0,  1.5,
  "1913–1950",         3.5,  1.5,
  "1950–1980",         4.0,  4.5,
  "1980–2010",         4.5,  2.5,
  "2010–2020",         5.0,  1.8,
  "Projection 2100",   4.0,  1.2
) |> dplyr::mutate(
  ecart = r - g,
  couleur = ifelse(ecart > 0, "r > g (concentration)", "g > r (compression)")
)

p <- plotly::plot_ly(df_rg, x=~periode) |>
  plotly::add_bars(y=~r, name="r (rendement capital %)", marker=list(color="#2563EB"),
    text=~paste0(periode,"<br>r=",r,"%")) |>
  plotly::add_bars(y=~g, name="g (croissance %)", marker=list(color="#059669"),
    text=~paste0(periode,"<br>g=",g,"%")) |>
  plotly::layout(
    barmode="group", xaxis=list(title=""), yaxis=list(title="Taux (%)"),
    hovermode="x unified", font=list(family="sans"),
    paper_bgcolor="rgba(0,0,0,0)", plot_bgcolor="rgba(0,0,0,0)",
    shapes=list(list(type="rect",x0=3.5,x1=4.5,y0=0,y1=5,
      fillcolor="rgba(5,150,105,.07)",line=list(width=0))),
    annotations=list(list(x=4,y=4.8,text="Trente Glorieuses\ng > r",showarrow=FALSE,
      font=list(size=10,color="#059669")))
  )
p
Tip

L’exception des Trente Glorieuses
C’est la seule période où g > r de façon durable. Piketty l’explique : destructions de patrimoine par deux guerres mondiales, fiscalité très progressive (jusqu’à 91% aux USA), syndicats puissants. Sans ces contre-forces politiques, r > g est la règle.

Questions de l'atelier 3
La période 1950–1980 est la seule où g > r. Quelles politiques ont rendu cela possible ? Depuis 1980, r > g est de retour. Quels instruments économiques permettraient de corriger cette tendance ? L'impôt mondial sur le capital proposé par Piketty est-il réaliste ?
