🔍 Détective des inégalités

Activité · Identifier une époque à partir d’indices économiques · +25 XP par énigme

Dans cette activité, tu joues le rôle d’un·e historien·ne qui doit identifier une époque et une société à partir d’indices économiques bruts. Pas de date, pas de nom — seulement des chiffres, des textes et des images. Mobilise ce que tu as appris dans les actes.

Tip

Conseils pour le détective
Les indices sont progressifs : les premiers sont larges, les derniers très spécifiques. Essaie de répondre tôt pour te challenger. Les réponses acceptent différentes formulations — époque, événement, pays, siècle.

    if (!input || !fb) return;
    var val = input.value.trim().toLowerCase()
      .normalize('NFD').replace(/[\u0300-\u036f]/g, '');
